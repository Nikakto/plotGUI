# plotGUI
GUI for matplotlib.
used:
+ PyQt4
+ pyperclip
+ matplotlib
+ numpy

Save default functions of matplotlib.plot and add:
+ copy/delete/paste objects (line, circle, rectangle, arrowPatch, text)
+ drag&drop for patches (circle, rectangle, arrowPatch, text)
+ wheel zoom
+ context menu for axes and objects
